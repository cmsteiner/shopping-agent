# Web Docs

This folder contains the planning and specification artifacts for the new interactive web interface.

## Files

- `requirements.md`: confirmed product requirements gathered interactively
- `software-spec.md`: functional spec, user stories, and acceptance criteria
- `technical-design.md`: recommended architecture and frontend/backend design
- `api-contract.md`: JSON and SSE API contract
- `database-schema-spec.md`: schema changes and migration plan
- `implementation-plan.md`: phased build plan
- `build-checklist.md`: milestone and ticket breakdown

## Recommended Reading Order

1. `requirements.md`
2. `software-spec.md`
3. `technical-design.md`
4. `api-contract.md`
5. `database-schema-spec.md`
6. `implementation-plan.md`
7. `build-checklist.md`
